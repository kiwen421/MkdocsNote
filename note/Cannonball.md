## 题目


贝西已经掌握了在长度为 $N$ $（1≤N≤10^5）$的数轴上变成炮弹并沿着数轴弹跳的艺术，位置从1到$N$从左到右编号。她从某个整数位置 $S$  $（1≤S≤N）$ 开始，以初始能量为 1 向右弹跳。如果贝西有能量 $k$，则她的下一次弹跳将在当前位置向前 $k$ 的距离。

1 到 $N$ 的每个整数位置都可以是目标或跳板。每个目标和跳板都有一个在范围 0 到 $N$ 内的整数值。值为 $v$ 的跳板会增加贝西的能量并反转她的方向。值为 $v$ 的目标将在能量至少为 $v$ 时被击碎，并且着陆时在目标上不会改变贝西的能量或方向。被击碎的目标将保持破碎状态，贝西仍然可以在其上弹跳，也不会改变能量或方向。

如果贝西弹跳无限时间或直到她离开数轴，她将击碎多少个目标？

如果贝西起始点在她可以击碎的目标上，她将立即执行击碎操作。同样，如果贝西起始点在跳板上，跳板的效果将在她的第一次跳跃之前应用。



## 输入描述

输入的第一行包含 $N$ 和 $S$，其中 $N$ 是数轴的长度，$S$ 是贝西的起始位置。 接下来的 $N$ 行描述每个位置。这些行中的第 $i$ 行包含整数 $q_i$ 和 $v_i$，其中如果位置 $i$ 是一个跳板，则 $q_i$=0，如果位置 $i$ 是一个目标，则 $q_i$=1，$v_i$ 是位置 $i$ 的值。



## 输出描述

输出一个数字，表示将被击碎的目标数量。





## 输入样例 #1

```cpp
5 2
0 1
1 1
1 2
0 1
1 1
```



## 输出样例 #1

```cpp
1
```





## 输入样例 #2

```cpp
6 4
0 3
1 1
1 2
1 1
0 1
1 1
```



## 输出样例 #2

```cpp
3
```





## 范围与解释

### 样例解释 #1

> 注意 1 是目标， 0是跳板

贝西从坐标 2 开始，那里是一个值为 1 的目标，所以她立即击碎它。然后她弹跳到坐标 3，那里是一个值为 2 的目标，所以她无法击碎它。她继续到坐标 4，这会改变她的方向并将她的能量增加 1 到 2。她弹回到坐标 2，那里是一个已经被击碎的目标，所以她继续。此时，她弹跳到坐标 0，所以她停止。她击碎了一个位于坐标 2 处的目标。

### 样例解释 #2

贝西的弹跳路径是 4→5→3→0→6，下一次弹跳将使她超出数轴（11）。她按顺序击碎了目标 4、3、6。



## 范围

- 输入 3-5：$N≤100$ 

- 输入 6-10：$N≤1000$

- 输入 11-20：无额外约束。
